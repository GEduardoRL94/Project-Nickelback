# Proyect-Nickelback
List of Nickelback songs and albums

1. Instalar XAMPP:
https://www.apachefriends.org/es/index.html

2. Iniciar servidores: 
  - MySQL Database
  - Apache Web Server
  
3. En el Navegador escribir la siguiente url:
  - localhost/phpmyadmin
  
4. Importar el Script "bd-usuarios.sql".
